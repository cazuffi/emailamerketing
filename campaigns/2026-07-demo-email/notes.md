# 2026-07-demo-email

## Quick workflow

1. Browse modules: `npm run preview -- catalog`
2. Add modules: `npm run add -- demo-email <module-id> <module-id>`
3. Build: `npm run build`
4. Preview: `npm run preview -- demo-email`
5. Paste `email.html` into D365 → Design → HTML
